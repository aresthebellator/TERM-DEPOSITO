# TERM-DEPOSITO

E' un semplice deposito con interfaccia CLI scritto puramente in python senza librerie esterne, utile per piccole aziende che hanno bisogno di un programmino per la gestione del proprio deposito.
Personalmente consiglio l'uso del programma su un server in caso ci fossero' piu' dispositivi da connettere in tempo reale.

# INSTALLAZIONE E AVVIO SU LINUX / MACOS
```bash
git clone https://github.com/aresthebellator/TERM-DEPOSITO.git
cd TERM-DEPOSITO
python3 term-deposito.py
```
